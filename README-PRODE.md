# Panel Web Admin PRODE

1. Instalar NodeJS si no lo tienes.
2. Abre la terminal en esta carpeta (`c:\Users\Roque\Documents\prode\admin-panel`).
3. Ejecutar `npm run dev` para iniciar servidor local.
4. Antes de usar: Edita el archivo `src/firebaseConfig.js` y coloca tus credenciales reales provistas por la consola de Firebase. Debes habilitar Firestore en la consola.
